# Hospital Analytics Dashboard

## Overview
An interactive Power BI dashboard for analyzing hospital operations and patient data.

## Features
- Total Patients
- Average Treatment Cost
- Total Revenue
- Payment Status
- Department-wise Revenue
- Monthly Revenue Trend
- Patient Analysis by Doctor
- Department Analysis
- Diagnosis Filter
- City Filter
- Gender Filter
- Department Filter
- Month Filter

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
